# Amazon Clone

## Overview
This is a responsive Amazon homepage clone built using HTML and CSS. The project recreates the basic layout and design of Amazon's homepage for learning frontend development.

## Features
- Responsive Navigation Bar
- Search Bar
- Hero Section
- Product Category Cards
- Footer Section
- Responsive Layout

## Technologies Used
- HTML5
- CSS3

## Project Structure

```
amazonCloneNew/
│── index.html
│── style.css
│── images/
```
